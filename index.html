I learnt that CSS or Cascading Style Sheets is a dynamic tool used in styling websites by formatting html elements to your desire. Css Box Model is the standard representation of an html element that is made up of the content, padding, border and margin. For best practice it's recommended to always reset the box-model of your website to standard using the css property {box-sizing: border-box;}. This would allow the height, width, border and padding you assign to an element's content to be included into the total height & width displayed by a browser.

The difference between paddings, margins and borders:

  • Padding is the space that's displayed on the inner side of an element which wraps around the content.
  • Border is a thick line that wraps around the padded element's content. It's displayed after the padding.
  • Margin is the space that's displayed at the outer side of an element's content. It wraps around the padding and border of an element.
